Module Hierarchy

  • commitmessage: A MVC framework for capturing commit information from SCM systems
    • attribute
    • controllers: Contains commitmessage.model.Controller implementations
      • cvs: The controller and utils for the CVS SCM (http://www.cvshome.org)
      • svn: The controller and utils for the Subversion SCM (http://subversion.tigris.org)
    • exceptions: Contains the CmException class
    • Itpl: String interpolation for Python (by Ka-Ping Yee, 14 Feb 2000).
    • model: Provides the model classes for the commitmessage framework
    • util: Basic utility functions and classes (CmConfigParser, getNewInstance, and execute)
    • views: Views work against the commitmessage.model to send out information to outside systems.
      • email: Provides an email commitmessage.model.Views
      • misc: A placeholder for future commitmessage.model.Views
      • post: Provides hooks into various systems via HTTP post (not implemented).

Class Hierarchy

  • commitmessage.Itpl.Itpl: Class representing a string with interpolation abilities.
  • commitmessage.Itpl.ItplFile: A file object that filters each write() through an interpolator.
  • commitmessage.model.Controller: A base implementation for SCM-specific controllers to extend; mostly does the generic Views handling.
    • commitmessage.controllers.cvs.CvsController: Translates CVS loginfo/commitinfo information into the model
    • commitmessage.controllers.svn.SvnController: Uses svnlook to pull data from svn repositories
  • commitmessage.model.Directory: A directory that has been affected by the commit
  • commitmessage.model.File: Represents a file that has been affected by the commit
  • commitmessage.model.Model: Wraps the Files and Directorys created by the Controller and used by Views
  • commitmessage.model.View: A base class for specific view implementations to extend
    • commitmessage.views.email.BaseEmailView: A basic email implementation for other style-specific email views to extend
      • commitmessage.views.email.ApacheStyleEmailView: Sends out an email in a style mimicking Apache commit emails (not implemented)
      • commitmessage.views.email.HtmlEmailView: Sends out an email in a style using HTML tables for better readability with variable-width font (not implemented)
      • commitmessage.views.email.TigrisStyleEmailView: Sends out an email in a style mimicking Tigris commit email format
  • exceptions.Exception: Common base class for all exceptions.
    • commitmessage.exceptions.CmException: The exception for all commitmessage-related exceptions
    • exceptions.StandardError: Base class for all standard Python exceptions.
      • exceptions.ValueError: Inappropriate argument value (of correct type).
        • commitmessage.Itpl.ItplError
